Desktop
========

.. include:: ../_static/badges/allplans-cloud-selfhosted.rst
  :start-after: :nosearch:

The Mattermost desktop app is available for Linux, Mac, and Windows operating systems. The Desktop App supports all the features of the web experience, and enable users to :doc:`connect to multiple Mattermost servers </preferences/connect-multiple-workspaces>` from a single interface, navigate between servers using keyboard shortcuts, and :doc:`customize their desktop user experience </preferences/customize-desktop-app-experience>`.

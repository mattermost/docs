Mattermost VM Image
===================

Use Mattermost Enterprise Edition VM image on your own cloud infrastructure.

This guide provides instructions for deploying a Mattermost-ready Linux VM in your cloud environment with Mattermost Enterprise Edition pre-installed and HA-oriented OS settings, leaving startup and configuration to your deployment process.

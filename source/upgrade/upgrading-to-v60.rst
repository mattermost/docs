Notes on Upgrading to Mattermost v6.0 or later
----------------------------------------------

Upgrading Mattermost desktop and mobile apps to the latest version is not mandatory when `upgrading to Mattermost Server v6.0 </upgrade/upgrading-mattermost-server.html>`__. Earlier 4.x versions of Mattermost desktop app, and earlier v1.x versions of the mobile app, will continue to be supported with a Mattermost v6.0 Server.
